0. 복습
1) transform, transition
   transform     요소의 크기조절, 이동, 회전 효과를 나타낼 때 사용한다
         display 속성이 block 또는 inline-block이어야한다

   translate(x, y)   넘겨준 값만큼 x, y 좌표 이동
   scale(x, y)   넘겨준 값만큼 확대, 축소(배수)
   rotate(각도deg)   넘겨준 값만큼 회전

   transition   변형에 시간을 설정하여 하나의 애니메이션처럼 표현할 수 있다
   transition의 속성
   transition-property : all | none | 속성
   transition-duration : 시간s
   transition-timing-function : 속성값   ease | linear | ease-in | ease-out | ease-in-out
   transition-delay : 시간s
   transition : property | duration | timing-function | delay

1. form 태그
1) textarea 태그
   <textarea cols="가로너비" rows="세로길이">내용</textarea>
   텍스트를 여러줄로 입력하는 영역을 만드는 태그

2) select, option 태그
   여러 옵션 중 하나를 선택하는 드롭다운 목록을 만드는 태그
   <select>
     <option value="값1">내용1</option>
     <option value="값2">내용2</option>
   </select>

   option의 value가 실질적으로 서버에 넘어가는 값이다
   seleted 속성을 이용하면 페이지 진입 시 선택되는 옵션을 설정할 수 있다

3) button 태그
   <button type="submit|reset|button">내용</button>
   버튼을 만드는 태그
   
   submit : 서버로 폼 데이터를 전송한다(기본값)
   reset : 폼 데이터를 리셋한다
   button : 딱히 기능 없이 버튼만 만든다, 주로 javascript와 함께 쓰인다

2. 반응형 웹
1) 반응형(responsive) 웹
   최근 웹 사이트를 pc뿐만 아니라 모바일, 태블릿 같은 기기로 접속하는 경우가 많다
   기기 화면에 맞춰서 새로운 사이트를 만드는 것은 비효율적이다
   그래서 사용자 화면 크기에 반응하는 사이트를 만들게 되었고 반응형 웹 사이트라고 한다
   반응형 웹 사이트는 기존 내용 그대로 유지하면서 사용자의 화면크기에 맞게 재배치하고 다시 표현한다

2) viewport(뷰포트)
   사용자 기기에서 실제 내용이 표시되는 화면 영역이다
   
3) 뷰포트 메타태그
   <meta name="viewport" content="width=device-width, initial-scale=1.0">

4) 뷰포트 메타태그의 속성
   width 
      뷰포트 너비 설정한다
      device-width 값을 주면 디바이스의 너비에 맞게 설정된다
   height
      뷰포트 높이설정한다
      device-height 값을 주면 디바이스의 높이에 맞게 설정된다
   initial-scale
      처음 페이지 진입 시 확대, 축소값을 설정한다
      1~10까지로 설정한다(기본값은 1)
   user-scalable
      확대, 축소 가능 여부를 설정한다
      yes, no로 설정한다(기본값은 yes)
   maximum-scale, minimum-scale
      최대, 최소 확대 비율을 설정한다

5) 뷰포트 단위
   vw(view width)
      100vw = 뷰포트 너비의 100%
   vh(view height)
      100vh = 뷰포트 높이의 100%
   vmin(view minimum)
      뷰포트 너비, 높이 중 작은 값을 의미함
      1vmin = 뷰포트 너비, 높이 중 작은값의 1%
   vmax(view maximum)
      뷰포트 너비, 높이 중 큰 값을 의미함
      1vmax = 뷰포트 너비, 높이 중 큰 값의 1%

6) 미디어 쿼리
   반응형 웹을 만들기 위한 가장 기본적인 방법이다
   사용자의 디바이스 뷰포트에 따라 다른 css를 적용할 수 있다

   @media 속성을 사용하여 특정 디바이스에 어떤 css를 적용할 지 설정한다

   @media [only | not] 미디어(디바이스)유형 [and 조건] * [and 조건]

   only, not
      특정 디바이스 유형만 적용시키거나 적용시키지 않을 디바이스를 설정할 수 있다(잘 사용되지 않는다)
   미디어유형
      여러 디바이스 유형을 설정할 수 있으나 보통 all 또는 screen을 사용한다
      all : 어떤 유형이든 다 적용한다(기본값)
      screen : pc나 모바일 유형을 의미한다
      ex) @media all {}
      디바이스 유형을 생략 후 바로 조건을 사용해도 된다
      ex) @media (width:400px){}
   and
      조건을 추가할 때 사용한다

3. css 우선순위
   속성 값 뒤에 !important를 붙인경우
      모든 우선순위를 무시하고 가장 높은 우선순위를 가지게 한다
   인라인스타일 적용한 경우
      html요소의 style속성으로 직접 지정한 스타일은 높은 우선순위를 가진다
   선택자로 id를 사용한 경우
      클래스나 태그선택자보다 높은 우선순위를 가진다
   선택자로 클래스, 속성, 가상클래스 등을 사용한 경우
      태그 선택자보다 높은 우선순위를 가지지만 id선택자보다는 낮은 우선순위를 가진다
   선택자로 태그, 가상요소를 사용한 경우
      가장 낮은 우선순위를 가진다

우선순위가 같다면 선택자를 상세하게 작성한 CSS가 적용된다
선택자도 동일하다면 CSS파일에서 나중에 작성된 규칙이 우선 적용된다
