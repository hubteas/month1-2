복습 -> html 태그 -> 인라인/블록요소 -> css

0. 복습
1) html / css / javascript
HTML(HyperText Markup Language)
	웹 문서의 구조를 정의하고 콘텐츠를 표현하는 기본 마크업 언어

CSS(Cascading Style Sheet)
	HTML 문서에 레이아웃과 디자인을 포함한 시각적요소를 정의하기 위한 규격

JavaScript
	이벤트 처리 또는 동적으로 변화하는 콘텐츠를 표현할 때 사용하며
	프로그래밍 언어이다

2) HTML 기본 구조
	!+enter
	
	<!DOCTYPE html>
	<html>
		<head>
			<title>제목</title>
		</head>
		<body>
		</body>
	</html>

3) html 태그
	<>를 사용하여 나타낸다.
	일반적으로 시작과 끝을 표현하는 2개의 쌍으로 이루어져있다.
	<시작태그 속성1="값1" 속성2="값2"...></종료태그>

4) 속성(Attribute)
	html 태그에 부가적인 정보를 제공하는 방법
	항상 시작태그 안에서 작성한다.

5) html 기본 태그
	<h1>~<h6>제목태그
	<p> 단락태그
	<strong>,<b> 굵은 글씨 표현하는 태그
	<em>,<i> 기울임 글꼴을 표현하는 태그
	<br> 줄바꿈 태그
	<hr> 가로줄을 그려주는 태그
	<ol> 순서가 있는 리스트태그
	<ul> 순서가 없는 리스트태그
	<li> 하부리스트태그
	<table> 테이블태그
	<caption> 해당 표의 제목태그
	<tr> 행을 만드는 태그
	<td> 열을 만드는 태그
	<th> 제목행의 열(컬럼)을 만드는 태그

1. 이미지 삽입 태그
	<img src="파일 경로" alt="설명">
	src 속성은 넣고 싶은 이미지의 경로를 입력하며, 웹 상의 주소를 입력해도 된다.
	alt 속성은 이미지에 대한 설명이며 만약 어떠한 이유로 이미지가 뜨지 않으면 설명이 대체되어 나온다.

2. 링크를 거는 태그(앵커태그, a태그)
	<a href="링크 주소나 경로 " > 링크를 걸어줄 요소 </a>
	a태그로 감싼 요소는 해당 경로로 링크가 걸린다.

3.  블록요소와 인라인 요소★★★
	요소를 여러 관점으로 분류할 수 있다
	그 중 가장 중요하게 봐야 할 부분이 블록요소와 인라인 요소이다.
	HTML의 모든 요소는 블록과 인라인으로 나뉜다.
	(CSS에서 인라인과 블록에 따른 처리를 하게 되므로 지금은 간단하게 알아본다)

1) 인라인 요소
	내용물(컨텐츠) 의 크기만큼만 영역을 차지하므로 다른 요소와 같은 줄에 위치할 수 있다.
	즉, 줄바꿈이 되지 않는다.
	대포적으로 span, img, strong, b, em,  등이 있다.

2) 블록 요소
	내용물(컨텐츠)과 상관없이 한 줄을 다 차지하므로 다른 요소와 같은 줄에 위치할 수 없다
	즉, 너비 속성이 100%를 다 차지하고 자동으로 줄바꿈이 된다.
	대표적으로 div, h1~h6, p 등이 있다.

4. CSS(Cascading Style Sheet)
	HTML로 만든 웹 문서의 스타일(디자인)을 담당한다.
	스타일이란 글꼴, 색상, 테두리, 정렬, 레이아웃(배치) 방법 등의 외관을 의미한다.
	HTML자체에서 제공하는 속성만으로도 간단한 스타일을 지정할 수 있다.
	그러나 내용과 디자인이 분리되어 있어야 유지 보수에 유리하므로 CSS 를 사용하는 것이 더 좋다.

1) CSS 형식
	속성1: 속성2;
	속성2: 속성2;
	속성은 미리 정해져있으며 필요에 따라 검색하여 사용한다.

2) 스타일을 적용하는 방법
	스타일을 적용시키는 방법은 인라인스타일, 내부스타일, 외부스타일 3가지가 있다.

(1) 인라인스타일
	스타일을 적용시킬 태그에 Style 속성을 사용한다.
	ex) <p style="color:red; background-color:yellow">내용</p>

	CSS 선언이 분명해서 다른 지정방식에 비해 요소들에 영향을 주고있는 CSS를 추적하기 쉽지만, 
	코드가 길어지고, CSS를 사용하는 의미인 HTML구조와 디자인의 분리라는 취지와 벗어난다.

(2) 내부스타일(Style 태그) 
	<head>영역에 <style></style> 태그를 사용하여 각 요소들의 스타일을 정의한다.
	선택자 {
		속성1: 속성값;
		속성2: 속성값;
	}

	인라인 스타일 방식에 비해 경제적으로 코딩할 수 있고
	html구조와 디자인 분리라는 CSS취지에 부합한다.

선택자 : 어떤 요소를 선택하기 위해 사용하는 것(태그, 클래스명, 아이디 명 등)
-id 와 class
	id와 class속성은 특정 태그들을 식별하고 그룹화하는데 사용한다.
	모든 태그에 사용가능한 속성이다.
	
	id : 특정 태그에 식별가능한 고유한 값을 부여한다.
		선택자로 사용할 때는 #id명으로 사용한다.
		(중복사용해도 오류가 발생하진 않지만, 중복으로 사용하는 것은 목적에 맞지 않다)

	class : 여러 태그를 하나의 그룹으로 묶어줄때 사용한다.
		선택자로 사용할 때는 .class명 으로 사용한다.

(3) 외부 스타일(외부파일 방식)
	별도의 CSS파일을 만들어 적용시킨다.
	HTML과 CSS를 완전히 분리하여 간리할 수 있기 떄문에 가장 많이 사용하는 방식이다.
	CSS파일을 HTML 파일에 적용시키기 위해서는 <head> 영역에 <link> 태그를 사용한다.

	<link rel="stylesheet" href="파일경로">
	rel 속성은 현재 페이지와의 관계를 의미하며	
	href 속성은 적용시킬 css파일의 경로를 적어준다.

	문법적으로는 css파일에서 사용할 때 style tag에 사용한 방식과 동일하다.
	파일의 교체하면서 디자인을 변경할 수 있다는 점에서 유지보수가 편리하다는 장점이 있다.
