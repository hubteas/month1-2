0. HTML/CSS/Javascript

HTML(HyperText Markup Language)
   웹문서의 구조를 정의하고 콘텐츠를 표현하는 기본 마크업 언어
   HyperText : 다른 텍스트에 대한 링크가 포함된 텍스트
   Markup Language : 텍스트에 의미를 부여하기 위해 문서에 표시하는 시스템

CSS(Cascading Style Sheet)
   HTML문서에 레이아웃과 디자인을 포함한 시각적 요소를 정의 하기 위한 규격
   HTML은 콘텐츠 내용과 구조를 정의하고 CSS에서 콘텐츠 위치, 정렬, 글자크기,
   여백, 배경색상 등을 정의하는 구조임
   디자인을 적용할 HTML 요소를 셀렉터(Selector)로 지정해 원하는 디자인 속성을 부여한다

JavaScript
   HTML 문서에 이벤트 처리나 동적으로 변화하는 콘텐츠를 표현하거나 서버와의 연결을 통해
   데이터를 가지고 오는 등의 프로그램적인 요소
   HTML은 텍스트 파일로 한 번 작성한 내용은 수정하기 전까지 항상 내용이 동일

1. HTML
   Hypertext Markup Language
   웹 문서를 만드는 언어

1) HTML 기본 구조
   <!DOCTYPE html>
   <html>      : 웹문서의 시작을 나타내는 태그
     <head>   
      이 페이지에 관련된 메타정보
      인코딩방식, 북마크에 설정될 이름 등
     </head>
     <body>
      본문 : 실제 웹 페이지에 나타나는 내용
     </body>
   </html>

2) HTML 문서에 사용한 4가지 태그
   <!DOCTYPE html> : 웹문서의 유형을 지정하는 선언문
      - 웹 브라우저에 현재 문서가 HTML5문서라고 알려주는 것이다
   <HTML> ~ </HTML> : 웹문서의 시작을 알리는 태그
   <HEAD> ~ </HEAD> : 웹 브라우저에 문서 정보를 알려주는 태그
      <head> 영역의 내용은 대부분 웹브라우저 화면에 보이지 않는다
      메타정보는 데이터에 관한 데이터를 의미한다
      <meta> 태그의 가장 중요한 역할은 화면에 글자를 표시할 때 어떤 인코딩방식을
      사용할지 지정하는 것에 있다
   <TITLE> ~ </TITLE> : 문서 제목을 나타내는 태그
      웹브라우저의 제목 표시줄에 표시, 해당 페이지의 방문자나 검색엔진은 제목표시줄의
      제목을 보고 페이지의 전체 내용을 추측할 수 있다
   <BODY> ~ </BODY> : 웹 브라우저에 내용을 표시하는 태그

3) HTML 태그
   태그는 <>를 사용하여 나타낸다
   태그는 일반적으로 시작과 끝을 표시하는 2개의 쌍으로 이루어져 있으며
   종료태그 앞에는 /를 붙여야한다

   <시작태그 속성="값" 속성="값">콘텐츠</종료태그>
   
   일부 태그는 닫는 태그가 없는 경우도 있다
   태그안에 태그가 들어갈 수 있으나 연 순서대로 닫아야한다
   
   주석은 <!-- -->의 형식으로 사용한다

4) 속성(Attribute)
   속성은 HTML 태그에 부가적인 정보를 제공하는 방법
   ex) 그림을 넣기위해 사용하는 <img>태그의 경우 그림파일의 위치를 지정하기 위한 속성 src속성
      <img src="C:\dog.jpg">
   항상 시작태그에 작성하며 속성명=값으로 구성

3. HTML 기본 요소와 태그
1)주석 
	<!-- -->
		단축키 : ctrl + /
2) HTML 기본요소
	<시작태그 속성1="속성값" 속성2="속성값" ...> 내용 </종료태그>

- 태그(Tag)
	<>로 표시한다
	일반적으로 <시작></종료> 한쌍으로 사용된다

-내용(Content)
	시작태그와 종료태그 사이에 있는 내용

-엘리먼트(Element)
	태그 + 내용
- 속성(Attribute)과 속성값(Value)
	태그<>안에 있는 일종의 옵션(설정)
	속성="속성값" 으로 사용
	여러 속성을 지정할 경우 각 속성을 띄어쓰기로 구분한다.

+) vs code에서 ! + enter 적용시 자동완성
<!DOCTYPE html>	: 현재 html 문서가 html5라는 것을 알려준다(버전)
<html lang="en">	: html 태그는 html의 시작을 알려주는 태그
		  lang="en" 이 페이지의 언어를 나타내는 속성
  <head>		: 이 페이지에 대한 정보(메타데이터)를 가지고 있다( 설명하는 정보들)
    <meta charset="UTF-8">	: 인코딩 타입
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
	: 뷰포트는 기기에 따라 어떻게 화면이 처음에 보일 지 정하는 속성값이다.
    <title>Document</title>	: 제목표시줄에 나타나는 제목
  </head>
  <body>	: 화면에 나타내고 싶은 정보를 작성하는 태그
  
  </body>
</html>

3) 태그의 종류
(1) <h1></h1> ~ <h6></h6>
	제목태그
	숫자가 작을수록 굵고 크게 표시된다.
(2) <p></p>
	단락태그
	단락은 위, 아래 간격이 벌어지며 자동으로 줄이 바뀐다.

(3) <strong></strong>, <b></b>
	굵은 글씨를 나타내는 태그

(4) <em></em>, <i></i>
	기울임을 나타내는 태그
(5) <br>
	줄 바꿈 태그
	닫는 태그가 없다(빈태그)
(6) <hr>
	가로선을 그려주는 태그
	닫는 태그가 없다(빈태그)

(7) 리스트 태그
	<ol></ol> : Ordered List의 약자, 순서있는 리스트를 만든다.
	<ul></ul> : Unordered List의 약자, 순서없는 리스트를 만든다.
	<li></li> : list 약자, ol 또는 ul태그 내부에서 사용한다.


(8) 테이블 태그
	<table></table> : 표를 만드는 태그
		테이블 태그는 표를 만들 때 사용하며 표의 제목, 행의 수 , 열의 수 등을 다른 태그를 이용해서 표현한다.
	<caption></caption> : 해당 표의 제목
	<tr></tr> : 행을 만드는 태그
	<td></td> : 열을 만드는 태그
		colspan, rowspan 속성
	<th></th> : 제목행에 컬럼을 만드는 태그
